# Imgutls - Image Utilities

A simple and intuitive web-based application for various image-related tasks. This tool provides a set of utilities to generate QR codes, create custom icons, and edit hero images directly in your browser.

## Live Demo

[Link to live demo](https://yabaggi.github.io/imgutls/)

## Features

### 1. QR Code Generator

- **Generate QR Codes:** Quickly create QR codes from any URL.
- **Customize:** Add a title and the URL as text below the QR code for context.
- **Download:** Save the generated QR code as a high-quality PNG file.

### 2. Icon Generator

- **Shapes & Sizes:** Create icons in square or round shapes with various standard sizes (e.g., 192x192, 512x512).
- **Gradients:** Apply beautiful linear gradients with customizable start and end colors.
- **Text Overlay:** Add text to your icon with options to change the font, size, color, and position (gravity).
- **Drawing Tools:** Add basic geometric shapes like lines, rectangles, circles, and polygons to your icon for more complex designs.
- **Live Preview:** See your icon update in real-time as you change the settings.
- **Download:** Get your final icon as a PNG file.

### 3. Hero Image Editor

- **Upload & Edit:** Upload any image to start editing.
- **Crop:** Easily crop the image by specifying the pixel values to remove from the top, bottom, left, and right.
- **Resize:** Define the exact output width and height for your image.
- **Optimize:**
    - Choose the output format: JPEG, PNG, or WebP.
    - Adjust the quality for JPEG and WebP formats to find the perfect balance between file size and visual quality.
- **Instant Feedback:** See the generated file size before you even download.
- **Download:** Save the optimized and edited image.

## How to Use

1.  **Open the App:** Open the `index.html` file in your favorite web browser.
2.  **Select a Tool:** Click on one of the three tabs at the top: "QR Code", "Icons", or "Hero Image".
3.  **Customize:** Use the intuitive form on the left to customize your image. The preview on the right will update instantly.
4.  **Download:** Once you are happy with the result, click the "Download" button to save the image to your computer.

## Technologies Used

- **HTML5:** For the structure of the application.
- **CSS3:** For styling and layout.
- **JavaScript (ES6+):** For all the application logic and interactivity.
- **[QRCode.js](https://github.com/davidshimjs/qrcodejs):** For generating QR codes.
- **HTML5 Canvas:** For all image generation and manipulation tasks.
